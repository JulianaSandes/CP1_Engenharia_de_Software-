# CP1_Engenharia_de_Software  
# 🍔 F-FOOD

---

## 📌 Descrição do Problema
Os pedidos na cantina são realizados de forma manual, tornando o processo sujeito a erros, atrasos e filas longas. Isso impacta diretamente os alunos, que possuem pouco tempo de intervalo, gerando frustração e perda de tempo. Além disso, os funcionários enfrentam dificuldades para organizar e processar os pedidos de forma eficiente.

---

## 🚀 Solução Proposta
O **F-FOOD** é um aplicativo mobile integrado ao sistema da FIAP que permite:

- Visualizar o cardápio da cantina  
- Selecionar múltiplos itens  
- Realizar o pagamento antecipado  
- Retirar o pedido sem enfrentar filas  

Com isso, o sistema reduz o tempo de espera, melhora a organização e aumenta a eficiência do atendimento.

---

## 🛠️ Tecnologias Utilizadas

- 📱 Frontend: Python
- 🌐 Backend: Python  
- 🗄️ Banco de Dados: Supabase
- 🧑‍💻 Versionamento: Git e GitHub  
- 🎨 Prototipação: Miro e Figma

## 🔧 Pré-requisitos
- **Python 3.10 ou superior:** Caso não tenha, baixe em python.org (Marque a opção "Add Python to PATH" durante a instalação).
- **Conexão com a Internet:** Necessária para validar o login e atualizar os pedidos em tempo real.

## ⚙️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/JulianaSandes/CP1_Engenharia_de_Software-.git
```

```bash
# Acesse a pasta
cd Projeto_F-Food
```

```bash
# Abra o seu Terminal ou Prompt de Comando (CMD), navegue até a pasta do projeto e execute o comando abaixo: 
pip install customtkinter supabase 
```

```bash
# *Nota: Se o comando acima falhar, tente usar:*
python -m pip install customtkinter supabase
```

## 📂 Estrutura do Projeto

```bash

├── backend/            # Contém a lógica de backend do projeto

```
